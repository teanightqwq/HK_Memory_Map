## 方法一：手動逐個添加

### 步驟 1：打開地圖
1. 前往 [Google My Maps](https://www.google.com/mymaps)
2. 找到地圖：「香港記憶拼圖」（地圖 ID: `1ntgBzVAnbWYyXTaxpRptN5ZMRJ7iJA4`）
3. 點擊「編輯」按鈕

### 步驟 2：添加地點
1. 點擊地圖上方的 **「添加標記」** 按鈕（📍圖示）
2. 在地圖上搜尋地址，例如：`中環結志街2號`
3. 點擊搜尋結果，標記會自動放置
4. 填寫標記資訊：
   - **標題**：`蘭芳園 - 絲襪奶茶`
   - **描述**：
     ```
     分類：餐飲系列 > 茶餐廳
     
     香港絲襪奶茶的代表，濃郁的紅茶配上淡奶，是最經典的港式飲品。
     
     地址：中環結志街2號
     ```

### 步驟 3：設定圖層（分類）
1. 在左側面板點擊「添加圖層」
2. 建議建立以下圖層：
   - 🍜 **餐飲系列**（紅色標記）
   - 🎭 **文化系列**（藍色標記）
   - 🏛️ **建築系列**（綠色標記）
3. 將標記拖曳到對應圖層

## 方法二：批量匯入（推薦！）

### 優點
- ✅ 一次匯入所有地點
- ✅ 自動分類
- ✅ 省時省力

### 步驟 1：準備 CSV 檔案

可以參考下列碎片的地點資料參考[地點資料整理](#地點資料整理) 區塊。

**CSV 檔案格式：**
```csv
名稱,地址,描述,圖層
蘭芳園 - 絲襪奶茶,中環結志街2號,分類：餐飲系列 > 茶餐廳。香港絲襪奶茶的代表。,餐飲系列
公和荳品廠 - 豆腐花,深水埗福榮街9-11號,分類：餐飲系列 > 甜品店。傳統豆腐花的經典味道。,餐飲系列
```

### 步驟 2：匯入到 Google My Maps

1. 打開地圖，點擊左側的 **「匯入」** 按鈕
2. 選擇準備好的 CSV 檔案（或直接從下方複製貼上到 Google Sheets）
3. Google 會自動識別地址欄位
4. 選擇「**地址**」作為位置欄位
5. 選擇「**名稱**」作為標記標題
6. 點擊「完成」

⚠️ **此時所有標記會在同一個圖層中！需要進行步驟 2.5 分組**

### 步驟 2.5：依圖層欄位分組（重要！）

匯入後，需要將標記依「圖層」欄位分組：

1. 找到剛匯入的圖層（通常叫「未命名圖層」或檔案名稱）
2. 點擊圖層名稱旁的 **⋯（更多選項）** 按鈕
3. 選擇「**依資料欄分組**」或「**Group places by column**」
4. 在彈出的選單中選擇「**圖層**」欄位
5. Google Maps 會自動分成 3 個圖層：
   - 餐飲系列
   - 文化系列
   - 建築系列

📸 **操作示意：**
```
左側面板：
┌─────────────────────────┐
│ 📍 未命名圖層           │
│   🎨 ⋯ 🗑️              │  ← 點這個 ⋯ (更多選項)
└─────────────────────────┘
        ↓
┌─────────────────────────┐
│ ○ 重新命名              │
│ ○ 依資料欄分組 ← 選這個│
│ ○ 刪除圖層              │
└─────────────────────────┘
        ↓
選擇欄位：[圖層 ▼]
        ↓
自動分成 3 個圖層 ✅
```

### 步驟 3：設定圖層樣式（重要！）

⚠️ **Google My Maps 不會自動使用 CSV 中的顏色**，需要手動設定：

#### 方法 A：依圖層統一設定顏色（推薦）

1. **匯入後會自動依「圖層」欄位分組**
   - 餐飲系列（一個圖層）
   - 文化系列（一個圖層）
   - 建築系列（一個圖層）

2. **為每個圖層設定顏色**
   - 點擊圖層名稱旁的 **🎨 油漆桶圖示**
   - 選擇「統一樣式」
   - 設定顏色：
     * 餐飲系列 → 🔴 紅色（Red）
     * 文化系列 → 🔵 藍色（Blue）
     * 建築系列 → 🟢 綠色（Green）
   - 選擇圖示：可選餐具🍴、文化🎭、建築🏛️等

3. **設定完成後記得儲存**

#### 方法 B：依資料欄位自動分配樣式

如果希望更細緻的顏色控制：

1. 點擊圖層名稱旁的 **🎨 油漆桶圖示**
2. 選擇「**依資料欄設定樣式**」
3. 選擇「**圖層**」欄位
4. Google Maps 會自動為每個值分配顏色
5. 手動調整每個值的顏色和圖示

#### 📸 操作示意

```
左側面板：
┌─────────────────────────┐
│ 📍 未命名圖層           │
│   🎨 ⋮ �️              │  ← 點這個油漆桶
└─────────────────────────┘
        ↓
┌─────────────────────────┐
│ ○ 統一樣式             │  ← 所有標記同一顏色
│ ○ 依資料欄設定樣式     │  ← 依欄位自動分配
└─────────────────────────┘
```

#### ✅ 最終效果

設定完成後，地圖上應該顯示：
- 🔴 紅色標記 = 餐飲系列
- 🔵 藍色標記 = 文化系列
- 🟢 綠色標記 = 建築系列

## 常見問題 FAQ

### Q1：匯入後所有標記都是藍色的？
**原因**：Google My Maps 不會自動使用 CSV 中的顏色欄位。

**解決方法**：
1. 點擊圖層名稱旁的 **🎨 油漆桶圖示**
2. 選擇「依資料欄設定樣式」→ 選擇「圖層」欄位
3. 手動為每個分類設定顏色：
   - 餐飲系列 → 紅色 #e74c3c
   - 文化系列 → 藍色 #3498db
   - 建築系列 → 綠色 #2ecc71

### Q2：匯入後所有標記都在同一個圖層？
**原因**：匯入時沒有依「圖層」欄位分組。

**解決方法**：
1. 點擊圖層名稱旁的 **⋯（更多選項）**
2. 選擇「**依資料欄分組**」（Group places by column）
3. 選擇「**圖層**」欄位
4. 會自動分成 3 個圖層：餐飲系列、文化系列、建築系列
5. 然後為每個圖層設定顏色

### Q3：地址無法識別怎麼辦？
- 嘗試在地址前加上「香港」，例如：`香港中環結志街2號`
- 使用更完整的地址格式
- 手動在地圖上搜尋並放置標記

### Q4：CSV 匯入時找不到「圖層」欄位？
**原因**：可能是 CSV 編碼問題或格式錯誤。

**解決方法**：
1. 確保 CSV 使用 UTF-8 編碼
2. 在 Excel 或 Google Sheets 中打開 CSV 檢查格式
3. 確認第一行是標題（名稱,地址,描述,圖層,分類顏色）
4. 如果仍找不到，匯入後手動依「圖層」欄位分組

### Q5：可以在標記上加圖片嗎？
- 可以！點擊標記後，在描述區域點擊 📷 圖示上傳圖片

### Q6：如何分享我的地圖？
- 點擊「分享」按鈕
- 設定為「公開」或「知道連結的使用者」
- 複製分享連結

### Q7：如何批量修改已匯入的標記？
- 無法直接批量編輯
- 建議：重新匯出修改後的 CSV，刪除舊圖層，重新匯入
